# 将地图导进我的世界

单人游戏：
进入 MC1_6_4\.minecraft\saves 文件夹中，将下载的地图存档复制进文件夹即可

bukkit 服务器：
将下载的地图存档复制进 Bukkit文件夹，打开Bukkit文件夹中的server.properties

#Minecraft server properties
#Wed Dec 19 19:56:36 GMT+08:00 2018
generator-settings=
allow-nether=true
level-name=shushan
enable-query=false
allow-flight=true
server-port=25565
level-type=DEFAULT

修改其中的level-name=地图名字，启动服务器即可导入。
### 中间不能有空格！
